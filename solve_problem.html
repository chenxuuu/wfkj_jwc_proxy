解决教务系统页面错位影响使用 bug 的方法：
原方法比较麻烦，点击此处查看旧方法
导致错位的原因是因为教务系统css写错了，原理不多解释，我就说下解决办法

本方法适用于chrome及所有webkit内核的国产浏览器（搜狗浏览器、百度浏览器、QQ 浏览器、UC 浏览器、遨游浏览器、360 极速浏览器、猎豹浏览器、2345 浏览器、Opera浏览器）、火狐浏览器
不适用于edge浏览器、IE及IE内核的国产浏览器（如360安全浏览器）。



首先安装一个插件：Stylish
插件一般在浏览器自带应用商店就能搜到，你也可以从此链接下载chrome版的Stylish插件：http://pan.baidu.com/s/1bo6QnF5
装插件不会的自己百度去吧
安装完毕之后点击插件，选择“管理已安装的样式。”


单击左侧的“编写新样式”：


在代码区域，复制粘贴填上：
.toolbox {
    margin-top: 100px;
}
.main_box {
    margin-top: 10px;
}

在下方的“应用对象”处，选择“该域上的网址”，在右边文本框中填入“wfkj.papapoi.com”。
为了兼容王瑞卿同学搭建的备用站wfxy.vuuuv.cn，我们需要再添加一个网址，选择“该域上的网址”，在右边文本框中填入“wfkj.papapoi.com”
在左边填好名称，点保存，如下图：


最后，问题解决。
